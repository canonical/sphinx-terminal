# Myst additional options

## Multi-line

```{terminal}
:copy:
:user: author
:host: canonical
:dir: ~/path

echo 'goodbye'
echo 'more'

goodbye
more
```

## No input

```{terminal}
:copy:
:user: author
:host: canonical
:dir: ~/path
:noinput:

hello
more
goodbye
```

## No output

```{terminal}
:copy:
:user: author
:host: canonical
:dir: ~/path

echo 'hello'
echo 'more'
echo 'goodbye'
```
