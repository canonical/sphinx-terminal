RST additional options
======================

Multi-line
----------

.. terminal::
    :copy:
    :scroll:
    :user: author
    :host: canonical
    :dir: ~/path

    echo 'more'
    echo 'even more!'

    more
    even more!

No input
--------

.. terminal::
    :copy:
    :scroll:
    :user: author
    :host: canonical
    :dir: ~/path
    :noinput:

    something
    something more

No output
---------

.. terminal::
    :copy:
    :scroll:
    :user: author
    :host: canonical
    :dir: ~/path

    echo 'something'
    echo 'something more'
